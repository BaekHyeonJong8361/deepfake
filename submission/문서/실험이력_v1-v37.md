# 딥페이크 탐지 모델 실험 이력 (v1~v37)

> 팀: 백현종(AI/딥러닝), 권민재(웹), 김지혁(아키텍처)
> 최종 OOD 평가셋: 140개 (SNS 화면녹화 real 70 / fake 70)
> 평가 지표: AUC-ROC

## Phase 2 — SNS 특화 (v19~v36)

**핵심 아키텍처 F3NetLiteV20**
```
입력 (299×299 RGB)
  ├─► RGB Branch (Xception backbone, 2048-d)
  └─► FAD 3-band Freq Branch (Xception backbone, 2048-d)
          ↓
  SEBlockFusion (Channel Attention + BatchNorm1d)
          ↓
  Linear(4096→1) → sigmoid
파라미터: ~44M
```

| 버전 | 베이스 | OOD AUC (140개) | val AUC | 핵심 변경 |

## 큰 흐름 요약

1. **v1~v5**: 아키텍처 기초 탐색 (Xception → F3NetLite → Freq-only)
2. **v6~v18**: in-distribution 최적화 → SNS OOD 0.411로 Phase 1 전면 실패 발견
3. **v19~v36**: F3NetLiteV20 기반 SNS 특화 파인튜닝 → 단일 모델 최고 **0.872 (v24)**
4. **앙상블 sweep** → 최종 **OOD AUC 0.8898** 달성 (v24+v25+v29 3-way)
